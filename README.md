# High Cloud Airline Analysis ✈️

## 📌 Project Overview
This project focuses on analyzing airline data to identify trends in flight performance, delays, and cancellations.

## 🛠 Tools Used
- SQL
- Excel
- Power BI

## 📊 Key Insights
- Identified major causes of flight delays
- Analyzed performance trends across routes
- Highlighted areas for operational improvement
